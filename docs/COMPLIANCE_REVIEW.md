# Biannual Compliance Review Process

This document outlines the process for conducting biannual compliance reviews for the Town of Wiley website.

## Scope
- ADA (Section 508, WCAG 2.1 AA)
- GDPR (General Data Protection Regulation)
- CCPA (California Consumer Privacy Act)

## Review Schedule
- **Frequency:** Every 6 months (January and July)
- **Responsible:** Compliance Officer or designated team

## Review Steps
1. **Accessibility Audit**
   - Run automated accessibility tests (Pa11y, Axe, Lighthouse)
   - Complete manual accessibility checklist
   - Test with screen readers and keyboard navigation
2. **Privacy & Data Protection**
   - Review privacy policy for accuracy and clarity
   - Verify cookie consent and opt-out mechanisms
   - Check data retention and deletion processes
3. **Security & Incident Response**
   - Review security audit logs and incident response plan
   - Confirm penetration testing is up-to-date
   - Validate backup and recovery procedures
4. **Documentation**
   - Record findings and remediation actions
   - File review report in compliance documentation
