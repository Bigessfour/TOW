Town of Wiley Admin Dashboard

How to Use This Admin Dashboard

  1. Choose a section below (News, Meetings, Officials, etc.) by clicking its box.
  2. Each section shows the latest information for that topic.
  3. To return to this menu, click the “Back to Admin Home” link at the bottom of any section page.
  4. If you need to add or update News or Pages, use the CMS Editor by clicking the button below.
  5. If you need help, contact your website support person at info@townofwiley.gov.
